# 7.1.4 LIVE PANNER

7.1.4 LIVE PANNER 是一款专为 Ableton Live 与 Max for Live 环境设计的沉浸式空间声像控制工具。

它并非传统意义上的“混音插件”，而更接近一台用于实时空间演奏与声音定位的数字控制台。整个系统围绕 7.1.4 多声道工作流构建，允许用户在二维平面与高度维度中自由移动声像，并通过实时的 Width、Focus、LFE 与 XYZ 控制，对声音的空间分布进行动态塑形。

不同于传统录音棚导向的软件逻辑，7.1.4 LIVE PANNER 更强调：

- 实时演奏性  
- 空间直觉  
- 清晰直接的控制逻辑  
- 低负担的人机交互  
- 空间声音作为“可演奏维度”的可能性  

整个插件采用适用于 Ableton Live Device View 的横向低高度布局，并针对 Max for Live 的实时性能与现场工作流进行了优化。它并不试图成为复杂庞大的空间混音系统，而更像一件用于沉浸式声音演出的空间乐器。

# 关于作者

7.1.4 LIVE PANNER 由北京音乐人、制作人、声音艺术家及现场音乐总监 Wang Lu（王璐 / L+R）开发与设计。

其工作长期横跨流行音乐制作、实验电子音乐、现场演出、多声道空间声音系统与沉浸式声音设计领域，并持续关注声音、空间、情绪感知与技术系统之间的关系。

在大型现场演出与主流音乐制作之外，他近年来的创作方向更多集中于：

- 沉浸式空间声音系统  
- 多声道现场演出  
- 空间音频乐器设计  
- Ambient / Healing 声音设计  
- 实验性交互界面  
- AI 辅助音乐生成与实时控制工具  

L+R SOUNDSYSTEM 是其当前长期发展的声音系统与实验性项目体系，关注“声音作为空间”而不仅仅是“声音作为内容”的创作逻辑。

7.1.4 LIVE PANNER 也是 L+R SOUNDSYSTEM 在空间声音控制工具方向上的首次系统化尝试。
