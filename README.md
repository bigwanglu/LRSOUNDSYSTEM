# L+R 7.1.4 LIVE Panner
L+R 7.1.4 LIVE Panner is a Max for Live spatial panner for Ableton Live, designed to route one audio track directly to 12-channel 7.1.4 speaker systems. It provides XY/Z positioning, Spread / Focus / LFE controls, 12-channel metering, output routing, and a built-in test signal for live performance and spatial audio setup.
